# ThemeLibrary
A very easy Android theme library

###Installation
Add it in your root `build.gradle` at the end of repositories:
````
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
        }   
    }
````
now add the dependency
````
dependencies {
    compile 'com.github.Iamsdt:ThemeLibrary:1.5'
}
````
